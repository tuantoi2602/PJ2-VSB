ASSIGNMENT
It is necessary to create classes, Contact and Address, you will be saved. Contact class will contain the name, age, email, weight and flag that the person lives and address. Address class will include street number, city and zip code.
Task 1 - XML Serialization
- serialize array of contact by XML serialization.  
- deserializer array of contact and print the results
- Load a saved contacts list and print in a console

Task 2 - Binary Serialization
- serialize array of contact by binary serialization.  
- deserializer array of contact and print the results
- Load a saved contacts list and print in a console

Bonus:
Ve tríde kontakt vytvorte eventu ZmenaVahy. V testovací tríde, která není serialozovatelná zaregistrujte u kontaktu obsluhu, reagující na tuto událost a daný kontakt se pokuste serialozovat. Nefunkcnost binární serializace vyrešte pomocí [field: NonSerialized].

Podívejte se prozkoumejte anotace trídních metod viz níže. Uvedomte si že pri vytvárení serializovaného objektu (binární serializace) se nevolá konstruktor.

[OnSerializing()]

[OnSerialized()]

[OnDeserializing()]

[OnDeserialized()]

Naopak u serializace pomocí XML je nutný konstruktor bez parametru.
